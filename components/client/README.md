# Client Portal Components

This directory contains components specific to the client-facing portal.

## Planned Components

- `OverviewContent.tsx` - Overview dashboard content
- `KeywordsContent.tsx` - Keywords analytics table and charts
- `CategoriesContent.tsx` - Categories performance breakdown
- `ProductsContent.tsx` - Products analytics
- `CoverageContent.tsx` - Coverage distribution charts
- `AuctionContent.tsx` - Auction insights and competitors

## Design Principles

- Clean, modern interface
- British English spelling throughout
- Brand colors: beige (#F2F1EB), gold (#F9B103), dark (#1C1D1C)
- Responsive design for all screen sizes
- Accessible (WCAG 2.1 AA compliant)
